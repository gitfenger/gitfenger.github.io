##安装阿里云服务器?
ubuntu16.04+PHP7+nginx

##安装步骤？

1. aliyun选择安装镜像(16.04.2 LTS)

2. 更新源 apt-get update

3. 查看安装的php版本

apt-cache show php

4. 安装php apt-get install php

5. 安装nginx apt-get install nginx

6. 修改nginx配置

  1. 定位到目录 cd /etc/nginx
  2. 编辑 vim nginx.conf

  3. 此文件有两行重要代码
    include /etc/nginx/conf.d/.conf;
    引入nginx配置文件
    include /etc/nginx/sites-enabled/    ;

  4. 配置虚拟主机的文件

7. 修改虚拟主机配置文件 /etc/nginx/sites-available

  1. 17行删除 default_server，这样可设置多个虚拟主机
  2. 18行是配置IPv6的，暂时用不到，删除
  3. 35行修改域名对应的文件夹 root /var/www/html/weixin;
  4. 38行添加index.php index index.php index.html index.htm index.nginx-debian.html;
  5. 40行二级域名 server_name weixin.genwoxueit.com;
  6. 50行，为了使nginx可以处理PHP脚本，需要开放以下配置

8. 如果需要设置多个虚拟主机

  1. 进入/etc/nginx/sites-available目录
  2. cp default default1
  3. 按照步骤7修改配置
  4. 设置软连接
    ln -s ../sites-available/default1 ./
